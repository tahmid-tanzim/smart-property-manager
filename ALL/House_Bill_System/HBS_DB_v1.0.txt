1. electricity_billing_rules
ebr_id---PK
ebr_start_unit [check not <= 0] / [for eg: 1]
ebr_end_unit [can be infinity, for e.g. >=400]/[check not <= ebr_start_unit]
ebr_per_unit_cost [check not <= 0]
ebr_strt_date [ month, year]
ebr_end_date [default max end_date for active]/[check not <= current row->ebr_strt_date]
ebr_rule_status [active/inactive] {Note: automatic active/inactive dependening on current_date exeeding end_date}

Note: for new rules new starting date will be same as previous end_date and new end_date will be max end_date.

wbr_strt_date [ month, year]
wbr_end_date [default max end_date for active]/[check not <= current row->wbr_strt_date]
wbr_rule_status [active/inactive] {Note: automatic active/inactive dependening on current_date exeeding end_date}

Note: for new rules new starting date will be same as previous end_date and new end_date will be max end_date.
